# 造 / Buildn

An AI-powered full-stack web application builder. Describe what you want, and Buildn creates it for you.

用自然语言描述你的想法，Buildn 帮你把它变成真实的应用。

## Vision

Buildn aims to make software creation accessible to everyone — no coding required.

## Tech Stack

- **AI Engine**: LLM-powered code generation
- **Frontend**: React + TypeScript + Tailwind CSS
- **Sandbox**: Secure code execution environment
- **Backend**: Supabase (Auth, Database, Storage)
